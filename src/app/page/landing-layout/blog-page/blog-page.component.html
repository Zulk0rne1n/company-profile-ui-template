What Is Open Template Hub

This article contains a guidance about all aspects of Open Template Hub.

What is Open Template Hub?

Open Template Hub is an Open Source Organization aims minimizing effort while building a new Software Business.

What does Open Template Hub provide to community?

We, as Open Template Hub Contributors, keep developing new functionalities and capabilities every day as Open Source. You can create your microservices, you can create your mobile and web applications which are integrated with those microservices, that means you can create your own business through Open Template Hub extremely fast.

Let's assume you have a new software business idea. Once you start implementation, you will need several common capabilities like Authentication, Payment, File Storage, Analytics, Basic Info Management and in the end, you will need a Web and Mobile GUI which are integrated with those capabilities, each and every time you have a new idea. Open Template Hub gives you that power to create your own business and make it alive just in a couple of minutes!

You can create your microservices using Server Generator:

Server Generator provides you the capability to generate 5 common-type microservices just with one command.

These are the service types you are able to generate for now:

Payment Server to get payment via Stripe, CoinBase and Paypal providers

Business Logic Server to manage your basic info operations

Authentication Server to manage authentications with supporting OAuth1 and OAuth2 flows (Social Logins with Facebook, Twitter, Github, etc..)

File Storage Server to store your files with S3 provider

Analytics Server to store your analytics data

You can easily generate these servers and you can deploy wherever you want; Heroku, GCP, Azure, AWS and so on. You will have also prepared Github Actions for many CI/CD ops once you deploy your repository to Github, Postman suites for TDD and attached to automated runs on Github Actions as well, stress tests and a lot of DevOps stuff are covered in these templates.

You can create your application interfaces using Application Generator:

Application Generator provides you the capability to generate 2 common-type application GUI just with a single command. These are the application types you are able to generate for now:

Web Application designed using Angular and integrated with all microservices above

Mobile Application designed using React Native and integrated with all microservices above

You will have lots of components, screens and API calls are covered for authentication, payment, file storage, analytics, basic user profiles and controllers integrated with services endpoints.

So, what's next?

We have so many plans to keep enhancing our functionalities, services and increasing external provider support. Implementing stream server for meetings and chat supporting significant providers like Zoom and Google, implementing MQ flow for all our servers, more components for user interfaces with enhancing UX, automated CI flows for builds and tests, automated deployment scripts using Terraform/Terragrunt or using some other proper tech to decrease build time of your business from minutes to seconds and so on..
